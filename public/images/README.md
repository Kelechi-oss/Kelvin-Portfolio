# Images

Drop your professional headshot here as **`headshot.jpg`** (or `.png`).

Until a real photo is added, the Hero section will show a placeholder card.
Once Sanity is configured, you can also upload the image directly through
the Studio and it will override this static asset.

Recommended specs:
- Aspect ratio: 3:4 portrait
- Min size: 720 × 960 px
- Format: JPG (quality 85+)
